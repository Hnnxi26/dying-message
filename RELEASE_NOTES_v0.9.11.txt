DYING MESSAGE v0.9.11 — 일러스트 제외 기능 완성본

반영 사항
- FINAL ROUND 시작 풀스크린 강조 연출
- 4라운드 카운트다운에 최종 추리 문구 표시
- 최종 추리를 메인 카드판에서 직접 선택
- 범인·도구·동기 각각 한 장만 선택 가능
- 최종 추리 제출 전 확인창
- 최종 추리 실패 즉시 안내 및 다시 수사하기
- 최종 실패 시 추리 타일 소모 및 선택 초기화
- 공통 힌트와 추리 타일 상단 sticky 배치
- 카드 영역 전체 폭 사용 및 카드 크기 확대
- 제외 카드는 뒷면으로 뒤집혀 뒤쪽에 배치
- 선택 카드는 확대·상승·체크 표시
- 프로젝터에서 최종 정답 카드 3장 순차 공개
- 프로젝터에서 조별 사건 해결/실패 결과 표시
- 브리핑 문구를 피해자의 다잉메시지 설정으로 수정
- 시민 → 시인 정정 유지

검증
- npm run build 통과
