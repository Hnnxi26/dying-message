# Dying Message Alpha

정식 프로젝트 구조로 다시 시작한 Alpha 버전입니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 접속

## 화면
- `/teacher` 교사 화면
- `/student` 학생 화면
- `/projector` 프로젝터 화면

## 현재 구현
- Next.js + TypeScript 구조
- 게임 데이터
- 게임 엔진 초안
- 교사/학생/프로젝터 화면 분리
- 힌트 타일 색상 규칙
- 카드 선택 UI 초안
- 자동 판정 로직 초안

Firebase 실시간 연동은 다음 Sprint에서 연결합니다.
