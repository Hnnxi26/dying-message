# Dying Message Rules v1.0

## 핵심 구조
- 교사는 소설가, 학생은 모두 추리자
- 모든 조는 같은 후보, 같은 정답, 같은 공통 힌트를 본다
- 조별로 다른 것은 제외 카드, 추리 타일, 제출 상태다

## 후보
- 범인 23명 중 9명
- 도구 23개 중 9개
- 동기 9개 전체

## 힌트
- 형용사 타일은 빨강
- 명사 타일은 파랑
- 공개된 타일은 학생과 교사 모두 본다
- 힌트로 사용된 타일은 중복 사용 불가
- 공개 전에는 수정 가능
- 공개 후에는 수정 불가

## 제외
- 학생은 범주 상관없이 6장 선택
- 정답 포함 시 제외 실패
- 실패 시 추리 타일 1개 소모
- 성공 시 카드 제외

## 라운드
- 모든 살아있는 조가 라운드를 완료하면 교사가 다음 라운드 시작
- 게임 오버 조는 다음 라운드 진행을 막지 않는다
